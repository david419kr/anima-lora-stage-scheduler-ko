# Anima LoRA 阶段介入

[English](README.md) | 中文

Anima LoRA 阶段介入是一个 Forge/WebUI 插件，用于控制 Anima LoRA 在采样过程中的介入时间。它可以让 LoRA 只在构图、人物特征或画风阶段生效，也可以为底图阶段和高分辨率修复阶段分别设置不同的调度。

## 功能

- 面向 Anima 模型的 LoRA 阶段介入，不修改模型文件。
- 内置构图阶段、人物特征阶段、画风阶段模板。
- 支持介入和禁止两种行为：让 LoRA 只在某阶段生效，或在某阶段强制为 0。
- 支持根据 Shift 自动调整介入时间。Shift 越高，窗口会轻微后移。
- 底图和高分辨率修复可分开设置。高分辨率修复可继承底图设置，也可使用独立面板。
- 高分辨率修复阶段可禁用全部 LoRA，或禁用指定完整 LoRA 标签。
- 支持单 LoRA 模板覆盖，同一提示词中的不同 LoRA 可使用不同阶段模板。
- 支持模板保存、重命名、删除，以及由基础预设组合生成自定义模板。
- 设置页提供界面语言切换：`zh` 和 `en`。

## 安装

1. 将插件文件夹复制到 `stable-diffusion-webui/extensions` 或 `sd-webui-forge-neo/extensions`。
2. 重启 WebUI，或使用 Reload UI。
3. 打开 txt2img 或 img2img，展开 **Anima LoRA 阶段介入** 面板。

## 基础使用

1. 勾选 **启用阶段介入控制**。
2. 选择目标范围：
   - 自动识别 Anima LoRA：只作用于检测为 Anima 的 LoRA。
   - 提示词中的全部 LoRA：作用于提示词内所有 LoRA 标签。
   - 仅目标列表：只作用于目标框内填写的名称。
3. 选择画风、构图或人物特征阶段模板。
4. 一般建议保持 **根据 Shift 自动设置介入时间** 开启。
5. 正常生成，插件会在采样期间动态调整 LoRA patch 强度。

## 时间参数

- **开始比例** 和 **结束比例** 是采样进度 `0` 到 `1`。
- **峰值比例** 是曲线最强的位置。
- **阶段强度倍率** 会乘到最终 LoRA 强度上。
- **强度曲线** 控制强度随采样进度变化的方式：平滑、保持、线性三角、前强后弱、后强前弱。
- **Shift** 会从 Forge 的 Shift / Distilled CFG 控件读取。自动模板会根据当前 Shift 调整窗口。

## 单 LoRA 模板

每行一个规则：

```text
lora_name=模板名
<lora:name:0.8>=模板名
```

没有填写规则的 LoRA 使用面板上方选择的模板。

## 模板管理

模板管理可以保存当前底图面板为模板、重命名自定义模板、删除自定义模板，也可以用基础阶段预设组合生成模板。内置模板不能覆盖、重命名或删除。

## 语言设置

进入 **设置**，找到 **Anima LoRA Stage Scheduler / Anima LoRA 阶段介入**，选择 `zh` 或 `en`，然后 Reload UI。

## 注意

- 为了兼容已有模板，内置模板 key 会保持稳定，所以英文界面下部分模板名仍可能保留中文。
- 插件只在采样时控制 LoRA patch 强度，不修改 checkpoint 或 LoRA 文件。
- 建议先用中等 LoRA 权重测试，避免同时堆叠过多强介入模板。
